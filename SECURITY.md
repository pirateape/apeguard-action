# Security Policy

## Supported Versions

ApeGuard follows [Semantic Versioning](https://semver.org/). Only the latest minor release receives security patches.

| Version | Supported |
|---------|-----------|
| latest  | ✅ |
| < latest| ❌ |

## Reporting a Vulnerability

This action ships ApeGuard binaries from GitHub Releases. If you discover a security vulnerability:

1. **Do not** open a public GitHub issue.
2. Email the maintainer directly or use GitHub's private vulnerability reporting.

ApeGuard's security model ensures no data exfiltration — all scanning runs locally on the runner.
